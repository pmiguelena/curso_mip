Clase 2

Paquete Tidyverse

A lo largo de esta clase, trabajaremos con el paquete Tidyverse.

Este conjunto de paquetes comparte una misma lógica de diseño y funciona en armonía para facilitar el análisis de datos.

Entre sus componentes, usaremos principalmente:

  • dplyr: para manipulación de datos (filtrado, selección, agrupación y mutación).

  • tidyr: para transformar y reordenar datos (pivotar, separar y unir columnas).

Para instalar y cargar Tidyverse, ejecuta:

## Paquete dplyr

El operador principal es %>% (pipe). Este toma el conjunto de datos a su izquierda y lo pasa implícitamente a la función de la derecha.

Crear nuevas columnas con mutate()

mutate() permite agregar una o varias columnas al data frame, basadas en operaciones sobre las variables existentes. Si el nombre coincide con una columna ya existente, su contenido se sobrescribe.

# Agregar columna 'maximo' que toma el valor máximo de t0
data <- aranceles %>%  mutate(
    maximo    = max(t0),            # Valor máximo
    minimo    = min(t0),            # Valor mínimo
    media     = mean(t0),           # Promedio
    suma      = sum(t0),            # Suma total
    rango     = max(t0) - min(t0),  # Diferencia máxima-mínima
    doble_t0  = t0 * 2              # Doble de cada valor de t0
  )

Resumir datos con summarise()

summarise() (o summarize()) permite calcular resúmenes estadísticos de una o más variables. Devuelve un data frame más compacto, útil para obtener valores agregados como el promedio, suma, máximo o mínimo. Puede aplicarse de forma global o en combinación con group_by() para obtener resúmenes por grupo.

_summarise() calcula estadísticas agregadas sobre todo el data frame o por grupos (cuando se usa junto a group_by()). Si se define el mismo nombre de columna existente, el resultado sobrescribe dicha columna.

# Resumen global: máximos, mínimos y promedio de t0
data_resumen <- aranceles %>%
  summarise(
    maximo_global = max(t0),
    minimo_global = min(t0),
    media_global  = mean(t0),
    suma_total    = sum(t0)
  )

data_resumen

1             5             1          2.5         30

Crear variables condicionales con case_when()

La función case_when() permite definir una nueva columna cuyos valores dependen de condiciones lógicas. Es útil para asignar etiquetas o transformar valores según múltiples criterios.

# Asignar valores según el país exportador 
data <- aranceles %>%   mutate(trade = case_when(    
  expo == "ARG" ~ 100,     
  expo == "BRA" ~ 200,     
  expo == "CAN" ~ 300,
  TRUE ~ 500))

Condiciones simples con if_else()

La función if_else() es útil cuando se desea aplicar una condición binaria (verdadero o falso) para generar una nueva variable.

# Crear una columna que indique si el arancel es cero o no 
data <- data %>%   mutate(arancel_cero = if_else(t0 == 0, "Sí", "No"))  # Mostrar las primeras 10 filas y nueva columna

Seleccionar columnas con select()

La función select() permite elegir un subconjunto de columnas de un data frame. Es útil para mantener solo las variables relevantes y descartar las innecesarias.

# Seleccionar solo las columnas 'expo' e 'impo' data <- aranceles %>%   dplyr::select(expo, impo)  # Mostrar las primeras 10 filas head(data, 10)

Ordenar filas con arrange()

La función arrange() ordena los datos según los valores de una o más columnas. Por defecto, el orden es ascendente. Se puede usar desc() para orden descendente.

# Ordenar las filas por la columna 'impo' 
data <- aranceles %>%   arrange(impo)  # Mostrar las primeras 10 filas y 2 columnas

Agrupar datos con group_by()

La función group_by() permite agrupar un data frame por una o más variables. Suele utilizarse en combinación con summarise() o mutate() para aplicar operaciones dentro de cada grupo.

# Agrupar por país exportador y ordenar por código hs6
data <- aranceles %>%
  group_by(hs6) %>%
  arrange(expo)

Agrupar y transformar con group_by() + mutate() + summarise()

Podemos combinar group_by(), mutate() y summarise() para realizar transformaciones y resúmenes dentro de grupos definidos por una o más variables.

Ejemplo: calcular promedio y clasificar

# Calcular el promedio por grupo y etiquetar si el valor está por encima del promedio de su grupo 
resultado <- aranceles %>%   group_by(expo) %>%   mutate(media_por_expo = mean(t0)) %>%   mutate(sobre_promedio = if_else(t0 > media_por_expo, "Sí", "No"))  %>% 
  ungroup() 
 
# Mostrar algunas filas
